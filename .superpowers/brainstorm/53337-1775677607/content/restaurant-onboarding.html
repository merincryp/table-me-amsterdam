Hoe komen restaurants in het systeem?

Je wilt richting honderden restaurants. Hoe vul je die database?

A

Handmatig toevoegen

Jullie voegen restaurants één voor één toe via een admin panel of direct in Supabase. Pro: volledige controle, kwaliteitsbewaking. Con: langzaam, veel werk.

B

Scrape Formitable restaurant-lijst

Formitable heeft een publieke lijst van aangesloten restaurants in Amsterdam. Scrape die als startpunt, vul aan met Google Places data (adres, foto, keuken). Pro: snel honderden restaurants. Con: data cleanup nodig, scraping kan breken.

C

Google Places bulk import + Formitable match

Haal alle restaurants in Amsterdam op via Google Places API, en match automatisch welke een Formitable widget hebben. Pro: breedste dekking, rijke data (ratings, openingstijden). Con: API kosten, veel data om te filteren.
