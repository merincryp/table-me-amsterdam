# Testing

## Manual Testing Checklist
- [ ] Mobile viewport (375px) — layout, text, navigation
- [ ] Tablet viewport (768px) — layout transitions
- [ ] Desktop viewport (1440px) — full layout
- [ ] Safari iOS — font rendering, smooth scroll
- [ ] Chrome Android — touch targets, viewport
- [ ] Dark mode (if applicable)

## Accessibility Testing
- [ ] Keyboard navigation (Tab, Enter, Escape)
- [ ] Screen reader: logical heading hierarchy
- [ ] Color contrast ratio >= 4.5:1 for text
- [ ] Focus indicators visible
- [ ] Images have descriptive alt text

## Performance
- [ ] Lighthouse score > 90 for all categories
- [ ] No render-blocking resources
- [ ] Fonts preloaded with `font-display: swap`
- [ ] Images optimized (WebP where possible)
