# Code Style

## HTML
- Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- Dutch `lang="nl"` attribute on `<html>`
- Self-closing tags with `/>`
- 2-space indentation

## CSS
- CSS custom properties (variables) in `:root` for theming
- Mobile-first responsive design
- No CSS frameworks — custom CSS only
- Logical property grouping: layout → box model → typography → visual

## JavaScript
- Vanilla JS only — no frameworks
- ES6+ syntax (const/let, arrow functions, template literals)
- Event delegation where practical
- No inline event handlers

## General
- All text content in Dutch unless specifically marked otherwise
- Keep everything in single `index.html` (inline CSS + JS) for simplicity
- Optimize for readability and maintainability
