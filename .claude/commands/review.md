# Review

Review the current state of the website for quality, accessibility, and performance.

## Steps
1. Read `index.html` fully
2. Check HTML validity and semantic structure
3. Verify responsive design patterns (mobile-first)
4. Check accessibility (ARIA labels, contrast, alt texts)
5. Review performance (image optimization, font loading, CSS efficiency)
6. Check SEO meta tags and Open Graph
7. Report findings with actionable items
